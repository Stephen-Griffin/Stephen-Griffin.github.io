    Movie Finder

    This is a movie finder I wrote in Python. The program asks if the user wants to find movies, tv shows,
    or both. The user can then choose to search on Netflix, HBO Max, or Disney Plus.
    Users can indicate if they have already seen the movie or show, and the program will
    not recommend it again. The movie or tv show will be json dumped into a file,
    and the user can see the list of movies or tv shows they have seen.
